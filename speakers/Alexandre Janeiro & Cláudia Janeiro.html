Resumo do seu Trabalho Final de Curso

Desenvolvimento de uma aplicação web C#, alimentada por uma base de dados SQL Server, para organização do processo de submissão e revisão de abstracts e papers, numa perspetiva de ambiente colaborativo. A aplicação web permite o registo e autenticação unívoca de utilizadores, funcionalidades role based (reviewers, technical committee, chairman), notificações, bem como a apresentação dinâmica de interfaces.

O objectivo deste trabalho foi o de aproveitar a experiência de ambos os alunos no meio académico para desenvolver um sistema de gestão de Conferências que incorporasse o know-how adquirido em relação ao modelo de negócio.
